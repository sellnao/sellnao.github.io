# sellnao.github.io

这是一个 GitHub Pages 静态站点的起始模板，包含简单的首页和样式。


快速开始：

1. 在本仓库中修改内容并提交到 `main` 分支。
2. 在 GitHub 仓库设置 -> Pages 中，将分支选择为 `main`（root）。
3. 等待几分钟，页面将会在 `https://sellnao.github.io/` 上可见。

建议的下一步：

- 添加 `about.html`、博客文章（`posts/`）或使用 Jekyll/Eleventy 生成器。
- 添加 `CNAME` 文件以绑定自定义域名。

示例 CNAME 文件：在仓库根目录创建 `CNAME`，内容为你的域名，例如：


```text
example.com
```

部署检查清单：

- 确保仓库已推送到 GitHub（`git push origin main`）。
- 在仓库设置 -> Pages 中启用 Pages，选择分支 `main` 和文件夹 `/(root)`。
- 若使用自定义域，添加 `CNAME` 并在 DNS 提供商处添加相应的 A/ALIAS 或 CNAME 记录。
